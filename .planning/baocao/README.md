# Bao Cao

Thu muc nay luu lich su lam viec va ket qua cac phase.

## Cau truc

- `sessions/`: bao cao theo ngay hoac theo phien lam viec.
- `phase-reports/`: bao cao tong ket theo phase hoac milestone da co bang chung.

## Quy tac viet report

- Ghi ro lenh da chay.
- Ghi ket qua that: pass/fail, so test/assertion neu co.
- Ghi loi con ton dong va nhom anh huong.
- Khong ghi `full suite pass` neu full suite chua pass that.

## Ghi chu di chuyen legacy

- `.planning/reports/20260421-session-report.md` -> `sessions/20260421-session-report.md`.
- `.planning/reports/20260422-session-report.md` -> `sessions/20260422-session-report.md`.
- Cac report co phase/milestone ro rang duoc chuyen vao `phase-reports/`.
- `phase-08-5-report.md` va `phase-08-6-report.md` la report legacy theo milestone that trong file cu.
- `phase-09-5-report.md` hien chi la placeholder `Chua xac minh` vi chua co report cu du bang chung.
