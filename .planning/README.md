# Planning Handover

`.planning` la nguon ban giao noi bo cho du an BMW Showroom & CRM Lead-Gen / Aftersales Platform.

Thu muc nay chi chua tai lieu du an, khong chua code runtime. Moi noi dung can trung thuc theo codebase, log va report that; thong tin chua ro phai ghi `Chua xac minh`.

## Tai lieu goc

- `PROJECT.md`: mo ta du an va cac quy uoc kien truc da ghi nhan.
- `REQUIREMENTS.md`: yeu cau nghiep vu.
- `ROADMAP.md`: lo trinh va task tiep theo.
- `STATE.md`: trang thai hien tai.
- `config.json`: cau hinh workflow tai lieu.

## Thu muc

- `baocao/`: bao cao phien lam viec va bao cao phase.
- `phases/`: ke hoach va tong ket tung phase.
- `bug/`: loi build, runtime, test va loi da sua.
- `kientruc/`: tai lieu kien truc he thong.

## Quy tac cap nhat

- Khong ghi `full suite pass` neu khong co log that.
- Khong bia phase hoac tinh nang chua co bang chung.
- Khong xoa report cu; neu di chuyen file da track thi dung `git mv`.
- Neu file cu khong khop cau truc moi, dat vao noi gan nhat va ghi chu legacy.
