# Bug Tracking

Thu muc nay ghi lai loi phat hien khi build, chay app hoac chay test.

## Cau truc

- `build/`: loi build/test/tooling.
- `runtime/`: loi khi chay app.
- `fixed/`: loi da xu ly theo phase.

## Quy tac ghi bug

- Ngay phat hien.
- Lenh hoac thao tac gay loi.
- Ket qua that.
- Nguyen nhan neu da biet.
- Workaround neu co.
- Trang thai: Open / Workaround / Fixed / Need Verify.
